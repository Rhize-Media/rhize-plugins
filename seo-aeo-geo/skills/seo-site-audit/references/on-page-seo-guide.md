# On-Page SEO Guide

Best practices for on-page SEO optimization.

## Title Tags

**Rules:**
- 50-60 characters (Google truncates at ~60)
- Primary keyword in first 3-4 words
- Unique across the entire site
- Compelling for click-through (not just keywords)
- Optional: brand name at end with ` | Brand` or ` — Brand`

**Formulas:**
- `[Primary Keyword]: [Benefit] | Brand`
- `[Number] [Adjective] [Keyword] in [Year]`
- `How to [Keyword] [Outcome]`
- `[Keyword] — [Unique Angle] | Brand`

**Common mistakes:**
- Duplicate titles across pages
- Keyword stuffing ("Best Tools Best Software Best Apps")
- Too generic ("Home" or "Blog")
- Missing entirely

## Meta Descriptions

**Rules:**
- 150-160 characters
- Include primary keyword naturally
- Include a call-to-action or value proposition
- Match search intent
- Unique per page

**Intent-matched patterns:**
- Informational: "Learn [what/how]. [Benefit]. [Supporting detail]."
- Commercial: "Compare [options]. Find [solution]. [Trust signal]."
- Transactional: "Get [product/service]. [Key benefit]. [Urgency/offer]."

## Heading Structure

**H1:**
- Exactly one per page
- Contains the primary keyword
- Matches the search intent (question for informational, solution for commercial)
- Should be similar to but can differ from title tag

**H2:**
- Major section headings
- Include secondary keywords
- Form a readable outline of the content
- Use question format for informational content ("What is X?", "How does X work?")

**H3-H4:**
- Subsection headings under H2s
- Long-tail keyword opportunities
- Never skip levels (H2 → H4 without H3)

## Content Optimization

**Keyword Integration:**
- Primary keyword in first 100 words
- Primary keyword in at least one H2
- 1-2% keyword density (natural usage)
- LSI/semantic keywords throughout
- Avoid keyword stuffing (> 3% density)

**Content Depth:**
- Analyze top 5 ranking competitors for target word count
- Cover all subtopics competitors cover (plus more)
- Include unique value: original data, expert quotes, tools, calculators

**Readability:**
- Short paragraphs (2-4 sentences)
- Subheadings every 200-300 words
- Bullet points and numbered lists for scannable information
- Active voice
- 8th-grade reading level for general audiences

## Internal Linking

**Density:** 3-5 contextual internal links per 1,000 words

**Anchor Text:**
- Descriptive (describes destination content)
- Contains relevant keywords naturally
- Varies across the site (not all exact-match)
- Avoid: "click here", "read more", "this article"

**Strategy:**
- Link from high-authority pages to priority pages
- Build hub-and-spoke clusters (pillar → supporting content)
- Link between related supporting content
- Add breadcrumb navigation
- Ensure no orphan pages

## Image Optimization

- Descriptive alt text (include keyword where relevant, but don't stuff)
- Compressed file size (WebP or AVIF format preferred)
- Explicit width and height attributes (prevents CLS)
- Lazy loading for below-fold images (`loading="lazy"`)
- Eager loading for above-fold hero images (`loading="eager"`)
- Descriptive file names (e.g., `project-management-dashboard.webp`)
